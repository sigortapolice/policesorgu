## Yayınlama (GitHub Pages)
1. Depoya dosyaları yükleyin (index.html ve README.md).
2. `Settings → Pages` → `Source: main` ve `/(root)` seçin.
3. Kaydedin. Site birkaç dakika içinde yayına geçecektir.
